# SSG Attendance Tool

A simple web-based attendance tool designed to help trainers and learners streamline attendance taking for MySkillsFuture funded courses in Singapore.

## Overview

The SSG Attendance Tool provides a quick and easy way to access QR code scanning and attendance status checking for SkillsFuture Singapore (SSG) funded courses. Users simply need to enter the 6-digit Course Run Code to access the necessary attendance functions.

## Features

- **Quick Course Code Entry**: Enter the 6-digit Course Run Code (without the "RA" prefix)
- **QR Code Generation**: Generate QR codes for attendance scanning
- **Attendance Status Check**: View whether attendance has been recorded

## How to Use

1. Open the application in your web browser
2. Enter the **6-digit Course Run Code** in the input field (e.g., if your course code is RA123456, enter only `123456`)
3. Once 6 digits are entered, two buttons will appear:
   - **Generate QR Code**: Opens the QR code page for scanning attendance
   - **View Attendance Status**: Opens the attendance status page to check if your attendance has been recorded

## Technical Details

- Pure HTML, CSS, and JavaScript implementation
- No backend required - runs entirely in the browser
- Responsive design for mobile and desktop use
- Integrates with MySkillsFuture.gov.sg attendance system

## Browser Compatibility

Works on all modern web browsers including:
- Chrome
- Firefox
- Safari
- Edge
